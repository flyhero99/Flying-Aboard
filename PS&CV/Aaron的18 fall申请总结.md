### 伪佛系申请者的DIY申请总结
```
每个申请者都会被失学的恐惧支配过，担心自己失学，整天诚惶诚恐，天天刷论坛，这都是常态。但
自我感觉，我的失学恐慌好像相当严重，过一段时间就会加学校，香港英国美国三处都有申请，最终
交了14个学校的申请，算下来申请费都交了不少。但依然觉得自己会失学，做好了gap的准备。要不
是前几天连中NWU和Upenn，自己毕设都可能做不下去吧。

回过头来看，我的整个申请季还是比较压抑的，首先是自己DIY特别没谱，然后因为实习期间比较忙，
前期学校基本都是踩ddl交上去的。好在研究院里有两个同届的CS申请者--Zoey和Hannah，三个人的
相互陪伴和鼓励，让整个申请季不再那么的痛苦。（其实当Hannah跟我们说她已经开始找工作时，我
整个人是懵逼的，明明才三月底，没有ad就开始找工作了嘛？？？整个人都不好了，然后就去加了一
个KCL）每次收到拒信的时候，我都会第一时间和两个同事以及NEU小分队的两个伙伴汇报。别人都是
报offer，但我报了半年的rej。这也算是我排解心中苦闷的一种方式了吧，要不是有人能倾听，我可
能会疯掉的。而面对最亲密的人，我一点也不敢跟他们说自己的申请情况，不想让他们操心。

申请者不会不知道415协议，在三月中的时候我还一个ad都没有，就特别担心自己会失学。从三月底
起我就进入精神衰弱的状态了：每天晚上睡不着，想刷一亩三分地，不到一两点不能入睡，第二天七
点多就会醒，中午午休也越来越难以入眠。尤其是如果入睡前看见地里开始发ad或者rej，基本一晚
上都睡不好了。第二天起来手哆哆嗦嗦的点开邮箱，发现也就一两个广告邮件。（这里尤其控诉USC，
MD，就他发ad和rej轮数最多，我至少有四个晚上都因为它睡不好觉，还有两个因为Rice没睡好）要
不是实习有事做，真不知道自己会成什么样子。当我申请季结束的时候，惊奇的发现，自己的发际线
啊，真特么高啊！！！年前明明不是这样子的呀！！！这里奉劝大家，最好申请之后就找个能让自己
全身心投入的事情做，不然你会疯掉的。论坛和邮箱，你刷或不刷，都不会影响你的结果，只会浪费
你大量的时间并让你心态越来越糟糕。想做到这一点，很难，但是，尽量吧。

有必要跟各位CS申请者说一下，面对AI的大热以及转专业申请人数的增加，CS的研究生申请真的已经
挤爆了！！！人数的增长带来最直接的影响就是，出结果的时间会变迟。所以大家一定要放平心态，
静候佳音。只要你还没有收到拒信，You're Still In the Game!(话说USC真的是申请体验极差，
  到现在都还没给我结果，再过几周春季offer都发完了都)。面对申请人数的激增，选校其实特别重
  要，给我的感觉就是，前20正统CS的申请为地狱难度。大家选校的时候一定要有点策略（手握顶会
  论文的大牛请绕行）。
```

##### 申请的前期准备
```
申请的前期准备无非是GPA，TOEFL/IELTS，GRE/GMAT，实习，科研等等。这些东西怎么
准备，我就不多说了，各个公众号什么的都有写，我之前的一些经验贴也都有提及。

我在这里想分享的，是本人从血和泪的教训（即前期让人怀疑人生的六连拒）里面总结出来的干货，
大家快拿起小本本记好。
```

1. GPA，TOEFL，GRE，实习/科研的权衡
```
最理想的情况，是大家各项全面发展，遍地开花。但人非大牛，孰能无过？这几个硬指标里面，难免
会有一两个拖后腿，在时间有限的情况下，该补哪个，是个需要考虑的问题

就我的经历和观察来看，GRE可以不到320，TOEFL万万万万不可以不过百（我这两组万万，看客自行
体会一下其重要性）。人们会一直觉得GRE比托福难，所以在托福GRE二选一的情况下会去刷GRE。但
是，语言成绩的重要性是远远高于GRE的。我觉得我前期的六连拒（尤其是Rice,NEU,OSU这三个学
校），很大程度上是由于我托福不过百所致，而托福不过百也限制了我的选校（前十五的我只选了一
个威斯康辛麦迪逊做彩票）。 所以奉劝大家，一定要好好刷托福，托福的重要性仅次于绩点，即是是
有交流经历都不能弥补（当初的我天真的以为凭着自己美国半年的交流经历能够把缺的一分给补上，但
提交NEU两周后就收到拒信的我明白了，只要标化成绩不过关，其他的东西小蜜是不会看的，你的简历
根本送不到申请委员会那边）

以上四个指标的重要性为 科研(能出优秀论文的那种)>GPA>TOEFL>实习/科研（一般的）>GRE。（仅
个人针对CS的意见）

重点：绩点托福很重要
```

2. 学校基本信息的考察

```
这个我也是血的教训。一般来说，学院的录取标准和申请截止日期，一般是比学校给出的要求更加严
格（绩点托福要求更高，申请截止日期会更早等）。我就是吃了这个亏，好几个学校我是打算提交的，
但是只看了学校的ddl没看学院ddl，导致申请交不上去。或者是只看了学校的申请要求没看学院的，
屁颠屁颠的看了半天，发现自己达不到要求，白survey了。（比方说UIUC，我记得CS的托福要求是
103及以上）。

所以给DIYer的建议是，先去学校的官网看看统一的截止日期和申请要求（一般都是托福和GRE），
要是够格，就去看学院的再check一下标准，否则就直接点击右上角关闭键然后看下一个学校吧。

推荐大家在申请开放前就能把自己想申请的“所有”学校，都过一遍信息，好心里有个数。这里给大家
安利一个软件“Teambition”，这是我实习期间的项目管理软件，后来突然发现可以创建留学申请的
模版，真的很方便管理申请的流程。网页端，PC端手机端都有。

重点：注意看学院的申请要求，推荐使用teambition
```

3. 项目的选择

```
这次申请给我的一个感觉是，CS和其他专业的申请难度完全不在一个水平线上。前20的正统CS录取
难度，我觉得还是很大的。最明显的，我申请专排20-40名左右的正统CS，除了收到一个西北大学的
录取，其他全拒，连专排40名的Boston和NEU都把我给拒了！！！但我申请宾大的CGGT就拿到了录取。

所以推荐各位CS申请者，别全CS，建议加几个MIS， Data, Security什么的，不为别的，就为能早
点拿个ad让你心静下来。我从12月份开始申请，直到四月底才拿到第一个ad（马里兰转专业录取我）
，5.8才拿到第一个CS录取。这漫长的半年我过的很艰难，如果再让我选，我一定会早早申请一个保
底让我不再恐慌。

还有就是，别只盯着CS学院下面的项目，据说有些学校ECE的项目，有些还是偏软的，可以通过后期
选课完成转码的操作。强烈建议大家多看看相似学院的其他项目。

重点：项目选择要多样化
```

4. 文书

```
相信文书是让很多DIY望而却步的一步，但回过头感觉，这个文书是最不重要的一个环节，没必要在
这个上面耗费太多精力。比较建议大家淘宝上找两个文书卖家，交叉润色一下即可。记住，千万别选
admitwriter！！！劳资买了他家的东西，感觉文书都没怎么给我改！！！辣鸡！！！坑钱货！知乎
上一堆人给它刷好评！！！辣鸡！！！辣鸡！！！辣鸡！！！

我十月份左右找了一个英国小哥给我改的文书，680，真的是物美价廉啊！遣词造句相当棒！后悔后
来找了admitwriter！

重点：文书淘宝解决，拒绝admitwriter！
```

5. 推荐信

```
好吧，其实如果你的推荐老师都是中国人的话，这个环节也没必要花太多精力了。因为大家都知道
推荐信是学生自己写的，没啥借鉴意义。推荐信自己写就好了，我就是自己写的。三四百词即可，
别整出啥语法错误即可。

如果能有外国老师给你推荐，那就是最好不过了。好好陶瓷，让他给你写好点。为了方便他书写，
你可以拟个草稿，方便老师回忆并借鉴一下你想让他写的几个点。这样即使他用不上，也会觉得你这
个人准备的很好。另外最好能早点确定选校名单，一次性把所有的链接都发给老师，不要申请一个
发一个，老师也会烦的（这点我做的不行，我后期加了好多次学校，感觉老师给我搞疯了）


重点：中国老师就自己写写即可，外国老师记得拟一个草稿，推荐信一次性发过去
```

6. 实习

```
强烈建议，刷完语言成绩再开始实习。我当初想着实习期间把托福给再刷一遍，后来发现自己太天
真了。国内IT行业加班是常态，在深圳经常加班到八九点，根本没时间学英语，所以最后也是拿着99
的托福跑去申请的，能中也是走了狗屎运了。

很多人可能想着等实习出个不错的结果再提交申请，这样文书和简历都会更有分量。这就是我当初
的想法，但后来发现，申请还是尽早提交比较好吧，最好是窗口一开放没多久就把申请交掉。后期
有一些成果了，发个邮件给招生处更新一下简历什么的可能比较好。（我的Rice，USC和OSU申请都
交迟了，如果能早点提交，不知道USC能不能给我一个消息）

比较建议在选校快结束时去实习。你已经用了一两周完成了学校的调研，也把文书，简历，推荐信什
么的搞得差不多了，最后只剩下填写网申表然后递交申请。在这种情况下，你去实习不会焦虑，能够
一心一意的去学习新东西。

重点：实习的时间要选好
```

7. 零零碎碎
```
最好的情况是大三暑假前GT已经刷完，找个实习做做事情。双休委屈一下自己，多做点学校方面的
调研，选个十来个学校，仔细查看他们的项目要求，准备资料，然后准备申请。等到个十月份左右
（针对美国），申请窗口开放，赶紧提交。

这里关于选校多说几句，我到3月份左右看到一个说法，觉得很正确。即关于彩票校，应该是有至少50%
的录取概率，毕竟招生老师也是宁缺毋滥的。高考第一志愿都填清北，也没啥意义。


重点：彩票校要有50%的概率
```
